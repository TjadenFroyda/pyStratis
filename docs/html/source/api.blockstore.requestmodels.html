BlockRequest¶

class api.blockstore.requestmodels.blockrequest.BlockRequest(*, Hash: pybitcoin.types.uint256.uint256, ShowTransactionDetails: bool = True, OutputJson: bool = True)¶

A request model for the blockstore block endpoint.

Parameters

  • hash (uint256) – The hash of the required block.

  • show_transaction_details (bool, optional) – A flag that indicates whether to return each block transaction complete with details or simply return transaction hashes. Default=True.

  • output_json (bool) – Output json or hex block. Default=True.
